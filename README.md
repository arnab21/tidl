tidl
====

Utilities for working with tidl files.

Code Status
-----------

[![Build Status](https://travis-ci.org/rravuri/tidl.png?branch=master)](https://travis-ci.org/rravuri/tidl)

Usage
-----

### using node

```shell
$ mkdir tidl
$ cd tidl
$ npm install https://github.com/rravuri/tidl/tarball/master
```
